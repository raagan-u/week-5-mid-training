Polling Application API

EndPoints

WebAuthn

  1. (POST) /auth/start-reg => To Start Registration for WebAuthn
  2. (POST) /auth/finish-reg => To Complete Registration for WebAuthn
  3. (POST) /auth/start-auth => To Start Authentication for WebAuthn
  4. (POST) /auth/finish-auth => To Complete Authentication for WebAuthn

Polling Data

  1. (POST) /poll/add-poll => To add poll data
  2. (GET) /poll/fetch-polls => To fetch all poll data
  3. (POST) /poll/update-poll => To update poll data
  4. (DELETE) /poll/delete-poll/{poll_id} => To delete poll data
